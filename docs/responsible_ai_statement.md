# Responsible AI Statement

This project is an analytics-first healthcare operations demo built with synthetic data only.

The AI copilot is not a clinician and must not diagnose, treat, counsel, or recommend care. It only summarizes structured operational metrics produced by SQL views and dashboard filters.

Allowed AI outputs:

- Executive summaries of access performance
- Referral delay explanations
- Clinic and ZIP-code bottleneck summaries
- No-show risk pattern summaries for operational outreach planning
- Data quality caveats

Disallowed AI outputs:

- Diagnoses
- Treatment recommendations
- Clinical triage decisions
- Patient eligibility decisions
- Any claim based on raw patient-level PHI
